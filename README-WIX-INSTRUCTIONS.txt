How to use in Wix:
1. Open wix-embed-full-page.html
2. Copy all the code
3. In Wix Editor: Add Elements > Embed Code > Embed HTML
4. Paste the code
5. Stretch the embed to full width
6. Increase the height until the full page shows correctly
